## üóÇ How to run the project

```bash

    # Clone the repository
    $ git clone https://github.com/Guntzel00/Github_Explorer.git

    # Enter in the repository directory
    $ cd Github_Explorer

    # Install the dependencies
    $ yarn install

    # Initialize the application
    $ yarn start
```
